## GDG Interactive Website

An immersive and experimental frontend project inspired by modern interactive web experiences.

Features
- Mouse-reactive 3D generative bar system
- Three.js interactive visualization
- Motion-responsive immersive UI
- Dynamic system-monitor inspired panels
- Creative frontend interaction design
- Experimental WebGL experience

### Tech Stack
Next.js / HTML
Three.js
JavaScript
Tailwind CSS
WebGL

### Focus
This project explores creative frontend engineering, interaction design and immersive user experiences.
