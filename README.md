## GDG Interactive Website

An immersive and experimental frontend project inspired by modern interactive web experiences.

### Features
- Mouse movement reactive animations
- Dynamic motion-generated visual bars
- Interactive landing page effects
- Smooth modern UI/UX transitions
- Responsive frontend design
- Creative experimental web interactions

### Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

### Focus
This project explores creative frontend engineering, interaction design and immersive user experiences.
